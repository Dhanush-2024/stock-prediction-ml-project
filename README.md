# Stock Prediction ML Project

This project focuses on predicting stock price trends using machine learning techniques.
It uses historical stock market data for preprocessing, feature engineering, and model training.

## Features
- Data preprocessing and feature selection
- Machine learning-based stock prediction
- Model evaluation and performance analysis
- Clean and modular project structure

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Objective
To apply machine learning concepts to real-world financial data and understand stock market prediction workflows.
